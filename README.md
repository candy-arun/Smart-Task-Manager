# Smart-Task-Manager


A simple yet powerful task manager built with **HTML, CSS, and Vanilla JavaScript** only.

## Features
- Add, mark complete/undo, delete tasks
- Input validation & empty list message
- Task counter
- Data persists using localStorage (page refresh safe)
- Responsive design


## How to Run
Just open index.html in any browser.
